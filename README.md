# StudentPerformanceCaseStudy
A Data-Driven Analysis of Academic and Lifestyle Factors Affecting Student Performance Using Python

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Tableau

## Dataset
15 student records with 12 attributes including 
Attendance, Study Hours, Mobile Usage, Stress Level etc.

## Key Findings
- Higher study hours = Higher marks ✅
- Higher mobile usage = Lower marks ❌
- Attendance strongly correlates with performance ✅
- Low stress students score significantly higher ✅

## Files
- analysis.py — Python code
- student_data.csv — Dataset
- output_screenshots/ — All charts
